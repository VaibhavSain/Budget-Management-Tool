# 💰 Budget Management Tool

A simple and interactive **Budget Management Tool** built using **HTML, CSS, and JavaScript** to help users track income, manage expenses, and calculate savings efficiently.

## 📌 Features

* 💰 Add and track income
* 📊 Record expenses by category
* 🧾 View total expenses
* 💡 Calculate remaining savings
* 🎯 Simple and clean UI
* ⚡ Fast and responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (Vanilla JS)** – Logic & DOM Manipulation

---

## ⚙️ How It Works

1. Enter your **total income**
2. Add **expense amount** and select a category
3. Click on **Add Expense**
4. Track total expenses dynamically
5. Click **Calculate** to see your savings
